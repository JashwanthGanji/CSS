Cascading Sytle sheets (CSS)

CSS (Cascading Style Sheets) is a language designed to simplify the process of making web pages presentable.

    It allows you to apply styles to HTML documents by prescribing colors, fonts, spacing, and positioning. The main advantages are the separation of content (in HTML) and styling (in CSS) and the same CSS rules can be used across all pages and not have to be rewritten.HTML uses tags, and CSS uses rule sets. CSS styles are applied to the HTML element using selectors.

There are Three ways to insert a style sheet into an html document :

  • Inline
  • Internal
  • External

Inline : An Inline style is used to apply a unique style for a single element

Internal : The internal style sheet may be used if one single

External : With an external style sheet, you can change the look of an entire website by changing just one file!

Selectors

CSS Selectors are used to select the html element you want to style

There are Five types of selectors we have :

  1. Simple selectors
    • Element
    • Class
    • Id
    • Grouping
    • Universal
  2. combinator
  3. Attribute
  4. Pseudo classes
  5. Pseudo elements

Simple Selectors

We have five types in simple selectors

Id Selector

The id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element! To select an element with a specific id, write a hash (#) character, followed by the id of the element.

Class Selectors

The class selector selects HTML elements with a specific class attribute. To select elements with a specific class, write a period (.) character, followed by the class name.

Element selector

Targets all elements of a specific type, such as paragraphs or headers. For example, setting a common font size for all paragraphs (P)

Grouping selector

Selects multiple elements by seperating each selector by comma(,) Ex: (p,.ab,#xy)

Grouping Selectors

The universal selector (*) selects all HTML elements on the page.
